# SecureUstuj - Менеджер паролей

### SecureUstuj - локальный менеджер паролей для безопасного хранения учетных данных. Приложение реализует полное шифрование данных, управление записями и генерацию паролей.

## Структура проекта

SecureUstuj/

├── Models/              # Модели данных

├── Services/           # Бизнес-логика

├── ViewModels/         # MVVM ViewModels

├── Views/              # Пользовательский интерфейс

└── SecureUstuj.db     # База данных

Мастер пароль ставится после первого ввода пароля.

Загруженно несколько тестовых данных.

Экспорт в CSV

Работает публикация в ClickOnce
